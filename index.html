Invoice Web Application

HTML5, CSS3, Sass, Bootstrap, Javascript, ASP.NET Core, Angular 6

This is a project I made for learning purposes. It involves an invoicing system, from which debtors can be added, and invoices can be made and sent out, or downloaded as pdf.

Debtors can sign in and view, download or pay their invoices.

In the backend, it makes use of a SQL database, which connects to a ASP.NET Core Web API using Entity Framework Core. For the front-end, an Angular based application is used, which makes HTTP requests with the Web API.

HINT Reservation System

HTML5, CSS3, Bootstrap, Javascript, ASP.NET Core, Angular 6, Raspberry Pi

This is a project my team and I made for school. It is an online web application, in which students and teachers can view schedules of available classrooms, and reserve one.

In the backend, it makes use of a SQL database, which connects to a ASP.NET Core Web API using Entity Framework Core. For the front-end, an Angular based application is used, which makes HTTP requests with the Web API.

A Raspberry Pi inside the classroom measures various data through sensors, such as temperature, light and sound levels.

Ovatic TicketSafe

Angular 7, Ionic, Typescript, ASP.NET Web API, C#

Graduation project for LVP Reserveringssytemen for finding a solution against ticket scalping. An app in which tickets can be kept, and secured by using encrypted dynamic QR-codes which activate two hours prior to an event.

GoGo Wallet

Angular 5, Ionic, Typescript, Node.JS, Raspberry Pi

Project for the minor Security Lab for creating an software application companioned by hardware. Hardware consists out of a self made wallet with a Raspberry Pi inside with GPS and Bluetooth, with an Ionic app. The wallet sends it current location to an API which saves it to a database. In the app users can track the location of their wallet in case of theft, and will receive a warning when they are about to leave their wallet behind.

HINT Reservation System

Angular 6, PostGres, ASP.NET Core Web API, C#, Raspberry Pi

Project for third year of the study to create a web application where students and teachers can reserve a room and view information such as the current temperature, sound level and luminosity level. These are being recorded by a Raspberry Pi containing various sensors.

Invoice Application

Angular 4, Typescript, ASP.NET Web API, C#

Project for learning purposes to create a web application in which invoices can be created. Debtors and products can be added to the system. PDF files can be generated based on these invoices. Debtors can sign in and view their invoices.
